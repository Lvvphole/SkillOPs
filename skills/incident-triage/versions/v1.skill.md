# Incident Triage Skill — v1

Classify high-severity or customer-impacting incidents as urgent; classify all
others as standard.

## Policy
- urgent  := severity == "high" OR customer_impact == true
- standard := otherwise
- action  := taken when label == urgent

## Notes
v1 baseline. Establishes the deterministic decision boundary and the evaluation
predicate used to compute success rate, false-positive rate, and action rate.
